# lexue_ddl
version:1.0<br>
主要功能：乐学通知事项的爬取<br>
登陆方法：cookie登陆<br>

使用方法：<br>
①登陆乐学 <br>
②按F12进入开发者模式 <br>
③进入Network选项 <br>
④找到"index.php" <br>
⑤找到"Request Headers" <br>
⑥找到"Cookie"键值对 <br>
⑦将其值复制填入到exp.py第21行中<br>
<strong>乐学cookie只能保持一周，一周过后需要更新cookie...</strong><br>
<sub>密码登陆功能暂时没做，如果有时间的话（划掉）继续更新</sub>
